# 剪贴板页面移动端优化方案

## 需要优化的地方

### 1. 搜索框和数量显示改为一行
- 搜索框：flex: 1，自动伸缩
- 数量显示：固定位置在右侧

### 2. 类型筛选改为标签页样式
- 样式参考导航页面的搜索引擎选择器
- 按钮：全部、文本、代码、链接、图片
- 移动端居中显示，支持横向滚动

### 3. 调整布局顺序
```
标题
━━━━━━━━━━━━━━━━━━━━━━
快速收集卡片
━━━━━━━━━━━━━━━━━━━━━━
[搜索框...................] [10条]
[全部][文本][代码][链接][图片]
━━━━━━━━━━━━━━━━━━━━━━
剪贴板列表...
```

### 4. 快速收集卡片底部按钮
```
[📋][🖼️][📤]              [保存]
```
- 左侧：三个图标按钮，间距紧凑
- 右侧：保存按钮
- 一行显示

## 实现步骤

1. 移动搜索和筛选到快速收集卡片下方
2. 搜索框和数量改为 flex 布局
3. 类型筛选改为标签页样式
4. 快速收集底部按钮改为左右布局
